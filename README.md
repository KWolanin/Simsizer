# Simsizer
Od graczki dla graczy - generator cech itp. dla polskiej wersji gry The Sims 4, przeznaczony głównie dla osób grających pokoleniowo i wszystkich potrzebujących urozmaicenia w rozgrywce. Obsługuje grę bazową i wszystkie do tej pory (09.01.20) wydane dodatki oraz umożliwia dopisywanie własnych cech sima do pliku txt.
